Goal-Based Investing (GBI) 는 "벤치마크 초과수익"이 아니라 "개인·기관의 구체적 재무 목표 달성 확률 극대화" 를 목적으로 하는 자산운용 패러다임입니다. 본 자료는 MPT의 한계에서 출발해 BPT·Mental Accounting·LDI·동적계획법·RL·로보어드바이저까지 12주에 걸쳐 이론·방법론·실무를 연결합니다.
