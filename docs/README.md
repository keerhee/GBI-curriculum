# docs/ — 학습 자료

이 폴더에는 **Goal-Based Investing (GBI)** 12주 커리큘럼의 모든 학습 자료가 있습니다.

## 파일 구성

- **`GBI_Curriculum_Syllabus.md`** — 12주 전체 syllabus (학습목표 · 평가 · 주차별 개요 · 참고문헌)
- **`Week01_Lecture_Note.md` ~ `Week12_Lecture_Note.md`** — 주차별 강의노트 12개
- **`WeekXX_Infographic_*.svg`** — 주차별 인포그래픽 24개 (주차당 2개)
- **`index.html`** — GitHub Pages 랜딩 페이지 (목차)

## 열람 방법

### 1) GitHub 웹

각 `.md` 파일을 GitHub 웹에서 바로 열람 가능합니다. 강의노트의 인포그래픽(SVG)과 LaTeX 수식 모두 GitHub의 기본 Markdown 렌더러에서 자동으로 렌더링됩니다.

### 2) Obsidian

1. Obsidian 실행 → **File → Open vault**
2. 이 `docs/` 폴더 선택
3. `GBI_Curriculum_Syllabus.md` 또는 `Week01_Lecture_Note.md` 부터 열기

### 3) VS Code

Markdown Preview 확장으로 열람. SVG 이미지는 프리뷰에서 자동 렌더링됩니다.

### 4) GitHub Pages

루트 레포의 Pages가 활성화되어 있다면 `https://keerhee.github.io/GBI-curriculum/` 에서 랜딩 페이지를 통해 네비게이션 가능.
