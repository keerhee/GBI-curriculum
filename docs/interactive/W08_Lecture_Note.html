В§2. DP кё°ліё мӣҗлҰ¬ вҖ” Bellman & Backward Induction

2.1 Bellman Equation

BELLMAN'S PRINCIPLE OF OPTIMALITY (1957)
$$V_t(s) = \max_{a \in \mathcal{A}(s)}\; \left[ r(s, a) + \sum_{s'} p(s'|s,a)\, V_{t+1}(s') \right]$$
Terminal condition: $V_T(s) = U(s)$. GBWMм—җм„ң $U(s) = \mathbb{1}\{W \ge G\}$.

2.2 Backward Induction м•Ңкі лҰ¬мҰҳ

1
Terminal: $V_T(s) = U(s)$ for all $s$
2
Backward loop: For $t = T-1, T-2, \ldots, 0$: For each $s$: $V_t(s) = \max_a [\ldots]$, $\pi_t(s) = \arg\max_a$
3
Optimal policy: $\pi = \{\pi_0, \pi_1, \ldots, \pi_{T-1}\}$

2.3 GBWMм—җм„ң DP мҡ”мҶҢмқҳ кө¬мІҙнҷ”

DP мҡ”мҶҢGBWMм—җм„ңмқҳ мқҳлҜё
мғҒнғң $s$$(W_t, t)$: нҳ„мһ¬ л¶ҖВ·мӢңм җ
Action $a$Portfolio м„ нғқ (efficient frontier мғҒ мң„м№ҳ)
Reward $r$мӨ‘к°„ 0, terminal $\mathbb{1}\{W_T \ge H\}$
Transition $p$GBM мҲҳмқөлҘ мқҳ мқҙмӮ° probability
Terminal $U$$\mathbb{1}\{W_T \ge G\}$

В§3. DORS2020 Framework вҖ” мғҒнғңкіөк°„В·кІ©мһҗВ·Backward Induction

3.1 л¬ём ң м„Өм •

мҙҲкё° мһҗмӮ° $W_0$, лӘ©н‘ң $G$, кё°к°„ $T$:
$$\max_{\{\pi_t\}_{t=0}^{T-1}}\; \Pr(W_T \ge G)$$

3.2 Efficient Frontier мң„ $K$к°ң Portfolio

н•өмӢ¬ лӢЁмҲңнҷ”: Action spaceлҘј frontier мң„ $K$к°ң (мҳҲ: $K=15$)лЎң м¶•мҶҢ. лӘЁл“  weight vector $w \in \Delta^n$ лҢҖмӢ . мқҙлЎ м ҒмңјлЎңлҸ„ near-optimalмқҙл©°, мӢӨл¬ҙ мҡҙмҡ©мӮ¬ model portfolio л©”лүҙмҷҖ м •нҷ•нһҲ мқјм№ҳ.

3.3 Wealth Grid (Log-Space)

$$W_i = W_0 \cdot \exp\!\left(\rho_{\text{grid}} \cdot i \cdot \sigma_{\min}\sqrt{h}\right)$$
GBM н•ҳ $\ln W$к°Җ м •к·ңл¶„нҸ¬ вҶ’ log-spaceк°Җ equal probability intervals. $\rho_{\text{grid}}$: grid density (DORS $m=15$ к¶ҢмһҘ).

3.4 Transition Probabilities

$$p_{ij}^{(k)} = \Phi\!\left(\frac{\ln W_{j+1/2} - \ln W_i - (\mu_k - \sigma_k^2/2)h}{\sigma_k\sqrt{h}}\right) - \Phi\!\left(\frac{\ln W_{j-1/2} - \ln W_i - (\mu_k - \sigma_k^2/2)h}{\sigma_k\sqrt{h}}\right)$$
log-normal л¶„нҸ¬мқҳ к°Ғ grid кө¬к°„ нҷ•лҘ .

3.5 Bellman Equation мҷ„м „нҢҗ

DORS2020 BELLMAN FOR GBWM
$$V_T(W_i) = \mathbb{1}\{W_i \ge G\}$$ $$V_t(W_i) = \max_{k \in \{1, \ldots, K\}}\; \sum_j p_{ij}^{(k)}\, V_{t+h}(W_j)$$ $$\pi_t^*(W_i) = \arg\max_{k}\; \sum_j p_{ij}^{(k)}\, V_{t+h}(W_j)$$

3.6 Python Pseudocode

def solve_gbwm_dp(W_grid, time_grid, portfolios, G, T):
    N = len(W_grid); M = len(time_grid); K = len(portfolios)
    V = np.zeros((N, M))
    policy = np.zeros((N, M-1), dtype=int)

    # Terminal
    V[:, -1] = (W_grid >= G).astype(float)

    # Backward recursion
    for t_idx in range(M-2, -1, -1):
        for i, W in enumerate(W_grid):
            best_v = -1; best_k = 0
            for k in range(K):
                mu_k, sig_k = portfolios[k]
                probs = compute_transitions(W, W_grid, mu_k, sig_k, h)
                v = np.sum(probs * V[:, t_idx + 1])
                if v > best_v:
                    best_v = v; best_k = k
            V[i, t_idx] = best_v
            policy[i, t_idx] = best_k
    return V, policy
