§2. VaR 제약의 수학적 재정식화

2.1 정의와 직관

$\alpha$-수준 VaR:
$$\mathrm{VaR}_\alpha(W) = -\inf\{w : \Pr(W \le w) > \alpha\}$$
chance constraint 형태:
$$\Pr(W < H) \le \alpha$$

2.2 정규분포 선형화

$W_T = W_0(1 + w^\top R),\ R \sim \mathcal{N}(\mu, \Sigma)$:
$$W_T \sim \mathcal{N}\!\left( W_0(1 + w^\top \mu), W_0^2\, w^\top \Sigma w \right)$$
$\Pr(W_T \le H) \le \alpha$를 전개:
$$\Phi\!\left( \frac{H/W_0 - (1+w^\top\mu)}{\sqrt{w^\top \Sigma w}} \right) \le \alpha$$
LINEAR VaR FORM — GBI 전체의 기초
$$(1 + w^\top\mu) - z_{1-\alpha}\sqrt{w^\top \Sigma w} \ge H/W_0$$
이는 $(\sigma_p, \mu_p)$ 평면의 직선 제약. Telser(1956) safety-first 제약선과 완전히 동일. 60년에 걸친 지적 계보를 단일 수식으로 수렴.

2.3 세 언어의 수학적 동일성

§3. Theorem — MVT–MA–VaR Equivalence

Theorem (Das, Markowitz, Scheid, Statman 2010)
자산 수익률 $R \sim \mathcal{N}(\mu, \Sigma)$, $\Sigma \succ 0$. Mental accounting 문제: $$\max_w \; \mu^\top w \quad \text{s.t. } \mu^\top w - z_{1-\alpha}\sqrt{w^\top \Sigma w} \ge \theta,\ \mathbf{1}^\top w = 1 \tag{MA}$$ 평균-분산 효용 문제: $$\max_w \; \mu^\top w - \tfrac{\gamma}{2}\, w^\top \Sigma w \quad \text{s.t. } \mathbf{1}^\top w = 1 \tag{MVU}$$ 이때:
  1. 임의의 $(\alpha, \theta)$에 대해, (MA) 해 = (MVU) 해인 $\gamma^*$가 존재.
  2. 역으로 임의의 $\gamma > 0$에 대해 구현하는 $(\alpha, \theta)$ 쌍 존재.
  3. (MA) 해는 MV 효율선 위.

3.1 증명 — Part (a): MA → MVU 매핑

1
제약 재배열. $\mu^\top w - \theta \ge z_{1-\alpha}\sqrt{w^\top \Sigma w}$. 양변 제곱: $(\mu^\top w - \theta)^2 \ge z_{1-\alpha}^2\, w^\top \Sigma w$.
2
Binding constraint. $(MA)$ 해에서 제약이 등호 성립 (아닐 경우 $\mu^\top w$를 더 키울 수 있어 모순): $\mu^\top w = \theta + z_{1-\alpha}\sqrt{w^\top\Sigma w}$.
3
Lagrangian (KKT).
$\mathcal{L}(w, \lambda, \eta) = \mu^\top w - \lambda[\mu^\top w - z_{1-\alpha}\sqrt{w^\top\Sigma w} - \theta] - \eta(\mathbf{1}^\top w - 1)$
4
$\partial\mathcal{L}/\partial w = 0$:
$(1-\lambda)\mu + \frac{\lambda z_{1-\alpha}}{\sigma_p}\Sigma w = \eta\mathbf{1}$
5
MVU FOC 비교. $\mu - \gamma\Sigma w = \tilde\eta\mathbf{1}$. 일치하려면:
$\gamma^* = \dfrac{\lambda z_{1-\alpha}}{(1-\lambda)\,\sigma_p^*}$
6
유효성. $\lambda \in (0, 1)$·$\sigma_p^* > 0$ → $\gamma^* > 0$. ∎

3.2 정리의 해석

TRIPLE EQUIVALENCE
$$\max \mu^\top w \; \text{s.t.}\; \Pr(W_T < H) \le \alpha \;\Leftrightarrow\; \max \mu^\top w - z_{1-\alpha}\sigma_p \;\Leftrightarrow\; \max \mu^\top w - \tfrac{\gamma^*}{2} \sigma_p^2$$
BPT(행동) = VaR(리스크) = MV(전통) — 세 언어, 하나의 수학적 대상.

§4. Aggregate Portfolio의 MV Efficiency

4.1 Theorem

Theorem · Aggregate MV Efficiency
조건: ① 자산 수익률 공통 $(\mu, \Sigma)$ 정규분포 · ② Short-selling 허용 · ③ 공통 asset universe.

이면 $w^{\text{agg}} = \sum_{k=1}^K \pi_k\, w_k^*$도 MV 효율선 위.

4.2 증명 — Convex Combination Argument

1
§3 Theorem: 각 $w_k^*$는 어떤 $\gamma_k^*$에 대한 MV 해.
2
Markowitz two-fund: $w_k^* = \frac{1}{\gamma_k^*} \Sigma^{-1}(\mu - \tilde\eta_k \mathbf{1})$. 모든 계정의 해가 $(\Sigma^{-1}\mu, \Sigma^{-1}\mathbf{1})$ span에 있다.
3
Aggregate: $w^{\text{agg}} = \sum_k \pi_k w_k^* = \frac{1}{\gamma^{\text{agg}}} \Sigma^{-1}(\mu - \tilde\eta^{\text{agg}} \mathbf{1})$.
여기서 $\frac{1}{\gamma^{\text{agg}}} = \sum_k \frac{\pi_k}{\gamma_k^*}$.
4
$w^{\text{agg}}$는 어떤 $\gamma^{\text{agg}}$에 대한 MV 효용 해의 형태 → MV 효율선 위. ∎
실무 함의: "Mental account를 쓴다고 효율성을 잃지 않는다" — 계정별 전혀 다른 $(H_k, \alpha_k)$ 설정(행동재무 유연성) + 전체 포트폴리오 MV 효율(이론 정합성). 1주차 §3.4에서 약속한 proof의 완성본.

4.3 Aggregation의 한계 조건

1
다른 자산 universe
계정 A는 US equity, 계정 B는 EM bond → aggregation 무의미
2
다른 시간지평
$T_1 \ne T_2$이면 $\mu_k, \Sigma_k$ 자체가 달라져 정리 기본 가정 깨짐
3
비정규 분포
Fat tail·skewness → VaR 제약이 MV 형태로 변환 안 됨
4
Dynamic rebalancing
정적 framework, multi-period는 W8 Das-Ostrov에서

§5. Short-selling 제약의 영향

5.1 Long-only MA 최적화 (SOCP)

$$\max \; \mu^\top w \;\text{s.t.}\; \|\Sigma^{1/2} w\|_2 \le \frac{\mu^\top w - \theta}{z_{1-\alpha}},\; \mathbf{1}^\top w = 1,\; w \ge 0$$
Second-order cone programming → cvxpy·scipy로 효율적으로 풀리는 convex program.

5.2 Das-Markowitz의 실증 관찰

Certainty Equivalent Loss: Short-sale 제약이 binding일 때도 효용 손실은 매우 작다(1–2 bp 이하). Normal 가정 하 short-sale 없이도 근접한 $(\mu_p, \sigma_p)$ 조합 달성 가능.

실무 함의: 한국처럼 공매도 제한 환경에서도 Das-Markowitz framework는 근사적으로 유효. 이론의 robustness.

5.3 Baptista(2012) Estimation Risk 확장

$\mu, \Sigma$가 추정치임을 고려하면:

  • Short-sale 없을 때 MA 해의 out-of-sample 성과가 MV보다 종종 더 좋음
  • 이유: MA는 $\alpha, H$로 직접적으로 위험 제어 → estimation error에 덜 민감
  • GBI는 단순한 "MV의 리브랜딩" 이상

6.3 Python 실습 스크립트

import numpy as np, cvxpy as cp
from scipy.stats import norm

mu = np.array([0.02, 0.09, 0.06])
Sigma = np.array([[0.0016, 0.001,  0.0005],
                  [0.001,  0.04,   0.01],
                  [0.0005, 0.01,   0.0225]])
L = np.linalg.cholesky(Sigma)

def solve_ma(H, alpha, short_sell=True):
    w = cp.Variable(3)
    z = float(norm.ppf(1 - alpha))
    cons = [cp.sum(w) == 1,
            mu @ w - z * cp.norm(L.T @ w, 2) >= H]
    if not short_sell:
        cons.append(w >= 0)
    cp.Problem(cp.Maximize(mu @ w), cons).solve()
    return w.value

w1 = solve_ma(-0.05, 0.05)   # retirement
w2 = solve_ma(-0.10, 0.10)   # education
w3 = solve_ma(-0.20, 0.20)   # bequest
w_agg = 0.6*w1 + 0.2*w2 + 0.2*w3

print(f"Aggregate: μ={mu @ w_agg:.4f}, σ={np.sqrt(w_agg @ Sigma @ w_agg):.4f}")
# Aggregate: μ=0.0767, σ=0.1475  (MV 효율선 위!)

§7. 한국 적용

7.1 MA 프레임으로 본 한국 상품

상품MA 관점 해석

개인투자자 공매도 제한 환경에서도 retail GBI 상품은 long-only SOCP로 구현. Das-Markowitz의 "short-sale 제약에서도 utility loss 작음" 결과가 핵심 robustness 보장. 2025년 국내 ETF AUM 200조 원+은 MA의 실무 구현을 크게 용이하게 했다.
