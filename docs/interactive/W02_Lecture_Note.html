SESSION 240min · Kahneman–Tversky

§2. Prospect Theory — Kahneman & Tversky

2.1 기대효용이론(EU)의 실증적 실패

von Neumann–Morgenstern(1944):
$$U(L) = \sum_i p_i\, u(x_i)$$

1960년대 이후 Allais·Ellsberg 등 다수의 실증 위반 축적. Kahneman-Tversky(1979) Econometrica에서 Prospect Theory로 4현상을 통합 설명:

  • Certainty effect: 확실한 결과에 과도한 가중
  • Reflection effect: gain/loss 도메인에서 위험선호 반전
  • Isolation effect: framing에 따른 평가 왜곡
  • Loss aversion: 손실 $100 ≈ 이익 $220의 심리적 충격

2.2 Prospect Theory의 두 축

AXIS 1
Value Function $v(x)$
$$v(x) = \begin{cases} x^\alpha & x \ge 0 \\ -\lambda(-x)^\beta & x < 0 \end{cases}$$
  • TK(1992): $\alpha = \beta \approx 0.88$, $\lambda \approx 2.25$
  • Gain concave → 위험회피
  • Loss convex → 위험선호 (reflection)
  • $\lambda > 1$ = loss aversion
$x$는 준거점(reference point) 대비 편차 — 이것이 EU와의 근본적 차이
AXIS 2
Probability Weighting $w(p)$
$$w(p) = \frac{p^\gamma}{\left(p^\gamma + (1-p)^\gamma\right)^{1/\gamma}}$$
  • $\gamma \approx 0.61$ (gains), $0.69$ (losses)
  • inverse-S 형상
  • 작은 확률 과대평가 → 로또
  • 큰 확률 과소평가 → 보험
로또와 보험이 동시에 존재하는 수학적 근거

2.3 Cumulative Prospect Theory (TK 1992)

rank-dependent utility를 차용한 누적 버전:
$$V(L) = \sum_{i=1}^n \pi_i^+\, v(x_i^+) + \sum_{j=1}^m \pi_j^-\, v(x_j^-)$$
여기서 $\pi$는 누적 확률에 $w$를 적용해 얻은 decision weight.

2.4 GBI 관점의 해석

준거점 = 목표금액 $H$: 준거점이 목표금액으로 설정되면, $W_T < H$는 "손실", $W_T > H$는 "이익". 목표 $H$ 기준 효용은 자연스럽게 kinked(꺾임) → 전통 MV 효용과 양립 불가능. 이것이 §3 MA, §4 SP/A와 같은 논리적 근원이다.

2.5 한국 실증 (자본시장연구원 2022)

관찰된 패턴
  • 여성 > 남성: 손실회피 강도
  • 30대 > 60대: 처분효과 강도
  • 신규·저분산 → 편향 심화
GBI 설계 함의
  • 세분화된 페르소나(age × gender × exp)
  • 목표 달성확률 % 중심 UX
  • 일간 평가를 숨기는 UI

SESSION 335min · Thaler

§3. Mental Accounting — Thaler

3.1 Thaler의 3요소

  1. Perception of outcomes — 결과를 어떻게 인식·평가 (framing)
  2. Assignment of activities to accounts — 활동의 계정 귀속
  3. Frequency of evaluation — 평가 빈도 (myopic loss aversion)

3.2 화폐의 비-대체성 (Non-Fungibility)

Thaler-Johnson(1990) "House Money"
  • 카지노에서 번 돈은 "하우스 머니" → 공격 베팅
  • 월급 vs 보너스 = 심리적으로 다른 돈
Prelec-Loewenstein(1998) "Painful Coupling"
  • 현금 결제 vs 신용카드 = 지출의 고통 차이
  • 선결제 휴가 → 사용 시점 "공짜" 심리

3.3 MA의 수학적 형식화

투자자는 부를 $K$개 계정으로 분할:
$$W = \sum_{k=1}^K W_k, \quad W_k = \pi_k W$$
각 계정은 고유의 효용함수:
$$U(W) = \sum_{k=1}^K \phi_k \cdot u_k(W_k - R_k)$$
$u_k$: 계정 $k$의 value function (PT 형태) · $R_k$: 준거점(=목표) · $\phi_k$: 중요도

3.4 MA의 두 얼굴 — 편향인가, 기법인가? (Brunel 2015)

Type A · Cognitive Bias (비합리)
  • 신용카드 리볼빙 18% 유지 + 적금 3.5%
  • 세금환급액을 "공짜돈" 인식 → 과소비
Type B · Rational Goal Allocation (합리)
  • 자녀 교육비 = 안전자산
  • 은퇴자금 = 분산투자
  • 여유자금 = 공격투자
GBI의 선택: B형 mental accounting을 의도적·명시적으로 설계. 편향의 이용이 아니라 구조의 선택이다.

3.5 한국 세제계정의 자연스런 MA

계정심리적 라벨자연스런 자산배분
IRP · 퇴직연금 DC"은퇴까지 건드리면 안되는 돈"TDF · 안정형
연금저축펀드"노후 보충용"중위험
ISA 중개형"세제혜택 받으며 굴리는 돈"주식·ETF 중심
일반 위탁계좌"자유롭게 굴리는 돈"공격적·단타
SESSION 440min · Lopes

§4. SP/A Theory — Lopes

PT vs SP/A: Prospect Theory가 "왜 목표지향적으로 사고하는가"를 설명한다면, SP/A는 "어떻게 목표지향적으로 사고하는가"의 수학적 구조를 제공한다. Shefrin-Statman(2000) BPT가 SP/A에 직접 기반한다 → 3주차의 전제.

4.1 Lopes의 두 요인(two-factor) 구조

Dispositional (성향)
  • Security-mindedness (S) — fear, 최악을 피하려는 성향
  • Potential-mindedness (P) — hope, 최선을 추구하려는 성향
Situational (상황)
  • Aspiration level $\rho$ — 달성하고 싶은 임계값
  • 목표 그 자체가 독립 decision criterion

4.2 SP/A의 수학적 정식화

SP 기준 — Decumulatively Weighted Value (He-Zhou 2012):
$$SP(X) = \int x\, d[-h(1 - F_X(x))]$$
두 성향의 혼합 가중함수:
$$h(z) = q_s\, h_s(z) + q_p\, h_p(z), \quad q_s + q_p = 1$$
$h_s$ convex (security), $h_p$ concave (potential). 결과적 $h$는 inverse-S.
A 기준 — Aspiration (Dubins-Savage 1976):
$$A(X; \rho) = \Pr(X \ge \rho)$$
통합 SP/A 목적함수:
$$V(X) = f\!\left(\, SP(X),\, A(X; \rho) \,\right)$$
실무 구현: aspiration이 제약으로 들어가는 형태:
$$\max_X \; SP(X) \quad \text{s.t. } A(X; \rho) \ge \bar{A}$$

4.3 SP/A vs Prospect Theory 비교

측면Prospect TheorySP/A Theory
준거점$x = 0$ (기점)도메인 전체의 aspiration $\rho$
위험태도효용함수 곡률decumulative weighting $h$
의사결정 축단일 (value·weight)이중 (SP · A)
실증 적합도—Lopes-Oden(1999) 실험에서 CPT보다 우수
GBI 함의value function 꺾임점 = 목표aspiration = 목표 그 자체
핵심: SP/A의 강점은 aspiration이 decision criterion의 독립 축이라는 점. GBI에서 목표금액 $H$는 bolted-on 제약이 아니라 의사결정의 핵심 축이 되어야 한다.

5.2 단일 수식으로의 통합 — BPT의 예고

UNIFIED GOAL-BASED VALUE FUNCTION
$$V(W; \mathcal{G}) = \sum_{k=1}^K \phi_k \cdot \mathbb{E}^h\!\left[ v_k(W_k - H_k) \cdot \mathbf{1}\{W_k \ge H_k\} \right]$$
$\phi_k$ = MA의 account weight · $v_k$ = PT의 kinked utility · $\mathbb{E}^h$ = SP/A의 decumulative weighted expectation · $\mathbf{1}\{W_k \ge H_k\}$ = aspiration 달성 지시함수

이 식이 Shefrin-Statman(2000) BPT로 이어지고, Das-Statman(2010) MA 프레임, Martellini(2015+) GBI formalization으로 확장된다 (W3–W4 주제).
"GBI는 교정 장치가 아니다" — 철학적 정립:
❌ 흔한 오해: GBI는 투자자의 비합리성을 수용해 차선의 해를 주는 것.
✅ 정확한 관점: GBI는 인간 의사결정 구조의 원래 형태를 수학적으로 formalize한 최적화 프레임이다. 편향의 수용이 아니라 구조의 수용.
