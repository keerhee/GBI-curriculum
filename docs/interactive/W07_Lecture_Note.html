§2. Multi-Goal 문제의 수학적 정식화

목표 집합:
$$\mathcal{G} = \{G_k = (H_k, T_k, \omega_k, \alpha_k, \chi_k)\}_{k=1}^K$$
$\chi_k \in \{\text{essential, important, aspirational}\}$: 포기 가능성
INTEGRATED MULTI-GOAL OBJECTIVE
$$\max_{\{w_k(\cdot), \pi_k(\cdot), \delta_k(\cdot)\}}\; \sum_{k=1}^K \omega_k \delta_k \cdot \Pr(W_{T_k}^k \ge H_k) + \sum_k \omega_k^{\text{up}} \cdot \mathbb{E}[U_k(W_{T_k}^k - H_k)^+]$$
제약: $\sum_k \pi_{k,t} W_t = W_t$ · $w_{k,t} \in \Delta$ · target-date 지출 $W_{T_k}^k \ge H_k$ 때 $H_k$ 지출

2.1 두 해결 전략

A · 분해 (Das-Markowitz 전통)
  • $\pi_{k,0}$ 외생 결정
  • 각 계정 독립 $w_k^*$
  • 주기적 $\pi_k$ 재조정
  • 상용 엔진 대부분
B · 통합 DP (Das-Ostrov 2022)
  • $(W_t, $ goals achieved$)$ 상태 변수
  • Portfolio + forgo 동시 최적화
  • Backward DP
  • 학술 진화 방향

§3. Funding Ratio Dynamics & 상태 공간

$$\mathrm{FR}_{k,t} = \frac{W_{k,t}}{F_{k,t}}, \quad F_{k,t} = \text{PV}_t(H_k, T_k)$$
Multi-Goal State Vector:
$$\mathbf{s}_t = (\mathrm{FR}_{1,t}, \ldots, \mathrm{FR}_{K,t}, T-t)$$

3.1 FR 동역학

$$\mathrm{FR}_{k,t+1} = \mathrm{FR}_{k,t} \cdot \frac{1 + w_{k,t}^\top r_{t+1}}{1 + r_{k,t+1}^F}$$
Asset return (제어 가능) vs Floor growth (시장 금리 의존)의 경쟁.

3.2 Aggregate Surplus

$$S_t = W_t - \sum_{k=1}^K F_{k,t}, \quad \overline{\mathrm{FR}}_t = \frac{W_t}{\sum_k F_{k,t}}$$
$S_t > 0$: self-sufficient · $S_t < 0$: 모든 목표 달성 불가능 → 일부 포기 필요

3.3 상태 공간 Action Regions (3-goal)

Region조건최적 Action
Region 1모든 $\mathrm{FR}_k > 1$De-risking + 잉여 분배
Region 2$\mathrm{FR}_j > 1$, $\mathrm{FR}_k < 1$$j \to k$ Transfer
Region 3모든 $\mathrm{FR}_k < 1$Triage (포기·축소)

§4. 목표 간 Rebalancing Rules

4.1 3가지 기본 형태

TYPE A
Scheduled (정기)
  • 고정 주기 (분기·반기·연간)
  • target allocation으로 복귀
  • ✓ 예측 가능, 단순
  • ✗ 최적 아님, 거래비용
TYPE B
Threshold-based
  • $|\mathrm{FR}_k - \mathrm{FR}_k^{\text{target}}| > \epsilon$
  • ✓ 거래 최소화
  • ✗ threshold arbitrary
TYPE C
Continuous (Martellini)
  • 매 시점 상태의존 정책
  • CPPI: $\alpha = m(W-F)$
  • ✓ 이론적 최적
  • ✗ 거래비용 고려 시 어려움

4.2 Surplus Transfer Rules

R1
Lock & Transfer
$\mathrm{FR}_j > 1.2$이면 초과분을 lock하고 잉여를 $\min \mathrm{FR}$ 목표로 이전: $\Delta = (\mathrm{FR}_j - 1.1) F_j$, $W_{k^*} += \Delta$, $k^* = \arg\min_k \mathrm{FR}_k$.
R2
Proportional Redistribution
모든 goal 간 FR을 균등화 방향으로 지속 이전.
R3
Priority-Weighted
높은 priority ($\omega_k$ 큰) 목표로 우선 이전.

4.4 Multi-Goal CPPI 확장

$$\alpha_{k,t}^{\text{PSP}} = m_k \cdot \frac{W_{k,t} - F_{k,t}}{W_{k,t}}$$
Essential goals: $m_k$ 작음(2–3). Aspirational: $m_k$ 큼(4–6). Chhabra 3-bucket의 동적 버전.

5.2 Das-Ostrov Goal Forgo Decision

GOAL FORGO DECISION VARIABLE
$$\delta_{k,t} \in \{0, 1\}: \text{goal } k \text{를 시점 } t \text{에서 포기할지}$$
Augmented objective:
$$V(\mathbf{s}_t) = \max_{\{w_t, \delta_t\}}\; \sum_k \omega_k \delta_{k,t} \cdot U_k$$
Bellman equation:
$$V(\mathbf{s}_t) = \max_{w, \delta}\; \sum_k \omega_k \delta_k + \mathbb{E}[V(\mathbf{s}_{t+1}) \mid \mathbf{s}_t, w, \delta]$$

5.3 Optimal Forgo Rule (informal)

$$\delta_k^* = 1 \iff \frac{\partial V}{\partial W_{k,t}} > \frac{\partial V}{\partial W_{j,t}} \cdot \text{transfer ratio}, \quad \forall j$$
"해당 goal의 marginal utility가 transfer 기회비용보다 클 때만 유지".

5.4 Fuzzy Threshold — 단계적 포기

Das-Ostrov 분석: Optimal forgo는 한 번에 이루어지지 않는다. FR 임계값 아래로 떨어질 때 단계적 포기.

Behavioral Challenge: 수학적으로 포기가 최적이어도 투자자는 감정적 거부 ("5년 준비한 여행이 이제 포기?"). 실무 처방:
① ISP 문서에 feasibility 가정·포기 우선순위 미리 명시
② Annual review에서 feasibility test 정기 검증
③ 위기 시 감정적 결정보다 사전 서약 rule 준수
