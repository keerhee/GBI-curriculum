§2. Telser Safety-First의 재조명

2.1 Safety-First의 역사적 뿌리 (Roy 1952 · Telser 1956)

1952
Roy — Safety-First Rule
$\min_w \Pr(R_p \le d)$. Markowitz와 같은 해(!)에 "재앙 수준 $d$ 이하 확률 최소화" 제안.
1956
Telser — 변형
$\max \mathbb{E}[R_p]$ s.t. $\Pr(R_p \le R_L) \le \varepsilon$. GBI의 원형이 이미 존재.
1963
Kataoka
$\max R_L$ s.t. $\Pr(R_p \le R_L) \le \varepsilon$.
2000
Shefrin-Statman BPT
Safety-First에 SP/A의 decumulative weighting 추가.
2010
Das-Markowitz-Scheid-Statman
Safety-First = VaR = MV의 등가성 증명.

2.2 Telser = GBI의 수학적 조상

Telser (1956)GBI (2015+)
$\max \mathbb{E}[R_p]$$\max \mathbb{E}[W_T]$
$\Pr(R_p \le R_L) \le \varepsilon$$\Pr(W_T < H) \le \alpha$
$R_L$: 참담한 수익률$H$: 목표 미달 수준
재발견의 역사: GBI의 chance-constrained 형태는 1956년에 이미 존재했다. MPT가 교과서를 지배하면서 50년 가까이 변방으로 밀려났을 뿐이다.

2.3 정규분포 가정 하 Telser 기하학

$$\Pr(R_p \le R_L) = \Phi\!\left( \frac{R_L - \mu_p}{\sigma_p} \right) \le \varepsilon$$
정리:
$$\mu_p \ge R_L + z_{1-\varepsilon}\, \sigma_p, \quad z_{1-\varepsilon} = \Phi^{-1}(1-\varepsilon)$$
$(\sigma_p, \mu_p)$ 평면의 반선(half-line). Slope: $z_{1-\varepsilon}$ · Intercept: $R_L$.

중요 통찰: Telser 문제의 해는 여전히 MV 효율선 위에 있다. Telser는 효율선을 거부한 것이 아니라, 효율선 위 어느 점을 고를 것인가의 기준을 "risk aversion $\lambda$"에서 "disaster avoidance $R_L, \varepsilon$"로 바꾼 것이다.

§3. BPT-SA (Single Mental Account) 최적화

3.1 정식화

$$\max_w \; E^h[W] \quad \text{s.t. } \Pr(W \le A) \le 1 - \alpha,\ w^\top \mathbf{1}=1,\ w_i \ge 0$$
여기서 $E^h[W] = \int_0^\infty [1 - h(F_W(x))]\, dx$ (decumulatively weighted expected wealth).

3.2 Lopes-Oden의 $h$ 함수

$$h(D) = \int_0^D \left[ q_s\, \phi_s(z) + q_p\, \phi_p(z) \right] dz$$
$\phi_s(z) = (1 + s_1)(1-z)^{s_1}$: security-minded · $\phi_p(z) = (1 + s_2) z^{s_2}$: potential-minded
성향$s_1$$s_2$$q_s$$q_p$
Security type2.922.250.790.21
Potential type0.770.230.160.84
Mixed1.81.20.50.5

3.3 BPT-SA 최적해의 정성적 특징 — "Bond + Lottery Ticket"

Shefrin-Statman Theorem
일정 조건 하 BPT-SA 최적 포트폴리오는 무위험채권 + 복권형 자산의 조합으로 표현된다.
🛡️ Bond 역할
  • $\Pr(W \ge A)$ 제약 만족시키는 hedge
  • Aspiration 기저를 채움
🎰 Lottery 역할
  • 왜도 큰 자산 · potential-minded 충족
  • Upside potential
중간적 자산 배제(Corner Solution): 분산된 주식 포트폴리오는 domination되어 포함되지 않는다. MV의 "모든 자산 일부씩 보유"와 정반대.

3.4 수치 예시 — 3-asset BPT-SA

설정
  • 무위험채권 $r_f = 0.02$
  • 주식시장 $r_s \sim \mathcal{N}(0.08, 0.2^2)$
  • 복권: 상태 $\omega_1$ 100 ($p=0.001$), 매입가 0.1
결과 (potential-minded)
  • BPT: $(w_f, w_s, w_l) \approx (0.85, 0.10, \mathbf{0.05})$
  • MV: $(0.4, 0.6, 0.0)$
  • 전혀 다른 모양

§4. BPT-MA — Multiple Account 피라미드

4.1 BPT-MA 정식화

$K$개 계정 $k = 1, \ldots, K$, 각각 aspiration $A_k$, 목표 확률 $\alpha_k$, 자금할당 $\pi_k$:
$$\max_{w_k} \; E^{h_k}[W_k] \quad \text{s.t. } \Pr(W_k \le A_k) \le 1 - \alpha_k$$
포트폴리오 집계:
$$w^{\text{aggregate}} = \sum_{k=1}^K \pi_k\, w_k^*$$

4.2 피라미드 구조의 발생

🚀
TOP Layer
부의 극적 확대
$\alpha \approx $ 20%
Lottery / 집중투자 · Potential-minded
📈
MIDDLE Layer
성장
$\alpha \approx $ 70%
Diversified Equity · Mixed
🛡️
BOTTOM Layer
빈곤 회피
$\alpha \approx $ 99%
Bond 위주 · Security-minded

4.3 두-펀드 정리가 BPT에서 깨지는 이유

1
각 계정의 $A_k, \alpha_k, h_k$가 다르다.
2
각 계정 최적해 $w_k^*$는 corner solution (bond + lottery)이며, 포함되는 자산 집합이 다르다.
3
aggregate $\sum_k \pi_k w_k^*$는 bond + 여러 종의 lottery의 조합.
4
이는 single-fund (market portfolio)로 수렴하지 않는다. CAPM과 결별.

§5. BPT 효율경계 vs MV 효율경계

5.1 Shefrin-Statman의 원 문장

"In general, the two frontiers do not coincide" — 일반적으로 일치하지 않는다. 그러나 언제 일치하고 언제 괴리되는지에 대한 완전한 분석은 후속 연구의 몫이었다.

5.2 Das-Markowitz-Scheid-Statman (2010) — 일치 조건

Sufficient Conditions for BPT-MV Coincidence
다음 조건 모두 만족 시 aggregate portfolio는 MV 효율선 위:
  1. 자산 수익률이 정규분포
  2. Short-selling 허용
  3. 각 계정의 제약이 VaR 형태

5.3 Pfiffelmann-Roger-Bourachnikova (2016) 실증

CRSP 1995–2011 실증:

결과
  • BPT 해가 MV 효율선 위: 70%+
  • BPT 최적: 고위험·고수익·양(+)의 왜도
  • MV의 10배 낮은 $\lambda$ 공격성
해석
  • BPT·MV는 같은 효율선을 공유할 수 있음
  • 그러나 선택 규칙이 다르다
  • BPT: 왜도·꼬리 민감 / MV: 분산 민감

5.4 언제 프론티어가 실질적으로 갈라지는가

1
Skewness 중요
자산에 유의한 왜도가 있을 때 (복권형·PE·옵션)
2
Tail fat
정규분포 가정 깨질 때 (극단적 market event)
3
Short-selling 금지
aggregate MV 효율성 보장 제약이 깨짐
4
Multiple disconnect goals
계정 간 rebalancing 불가 시
함의: 이 네 조건이 GBI가 특히 효과를 발휘하는 환경의 지도다.

§6. 한국 사례 — ELS와 "채권 + 복권" 구조

한국의 주가연계증권(ELS)은 Shefrin-Statman이 예측한 "bond + lottery ticket" 구조의 상품화 사례.

6.1 스텝다운형 ELS의 BPT 해석

$$W_T \approx \begin{cases} W_0 (1 + c T) & \text{조기상환 } (\Pr \approx 80\%) \\ W_0 (1 + c T') & \text{만기 쿠폰 } (\Pr \approx 15\%) \\ W_0 (1 - \ell) & \text{knock-in 손실 } (\Pr \approx 5\%) \end{cases}$$
$A = W_0(1 + cT)$: aspiration, $\alpha \approx 0.95$. 극단 tail의 knock-in = put option 매도 프리미엄이 쿠폰 재원.
심리적 매력: $\alpha=95\%$ 성공확률 → security-minded 충족. 채권 대비 2-3배 높은 쿠폰 → potential-minded 충족. 완벽한 BPT 상품 구조.

6.2 2024년 H지수 ELS 사태 — BPT가 예측하는 실패

실패 모드: 투자자 인식 "원금 거의 보장" ($\alpha=95\%$ aspiration 실현) vs 실제 "5% tail이 한꺼번에 실현" → 수조 원 손실. 투자자가 저확률 tail을 과소평가(probability weighting 실패).

6.3 GBI 관점의 처방

✅ ELS 상품의 올바른 편입 방식
Essential goal 버킷에 ELS 편입 금지 — tail risk가 $\alpha=99\%$와 양립 불가
Aspirational 버킷에만 — tail 실현도 감내 가능한 여유자금
목표 달성확률 명시 — "95% 확률" vs "원금보장"은 전혀 다른 프레임

6.4 한국 연금 3층 구조 = BPT-MA 피라미드

층상품BPT 대응
1층국민연금·기초연금Bottom ($\alpha \approx 99\%$, bond-like)
2층퇴직연금 DC·IRPMiddle ($\alpha \approx 80\%$, TDF·분산)
3층개인연금·ISA·일반계좌Top ($\alpha \approx 30$–$50\%$, 성장·집중)
