§2. Das-Varma (2020) — Q-learning GBWM

2.1 Q-learning Update Rule (Watkins 1989)

$$Q^*(s, a) = \mathbb{E}\!\left[ r_t + \gamma \max_{a'} Q^*(s_{t+1}, a') \mid s_t = s, a_t = a \right]$$ $$Q(s, a) \leftarrow Q(s, a) + \alpha [r + \gamma \max_{a'} Q(s', a') - Q(s, a)]$$

2.2 DP 동등성 (Watkins-Dayan 1992)

Convergence Theorem
tabular Q-learning은 $\sum_t \alpha_t = \infty$ + $\sum_t \alpha_t^2 < \infty$ 조건 하 $Q^*$로 수렴.
Das-Varma 2020 실증: DP $V(100, 0) = 0.627$ vs Q-learning $\approx 0.620$–$0.625$ (10⁶ episodes 후). 차이 < 1%p.

2.3 Training Loop Python

for episode in range(N_episodes):
    W, t = W_0, 0
    while t < T:
        # ε-greedy
        if random() < epsilon:
            k = random.randint(0, K-1)
        else:
            k = argmax(Q[s])
        # GBM simulation
        W_next = W * np.exp((mu[k] - sig[k]**2/2)*h + sig[k]*np.sqrt(h)*Z)
        r = 1 if (t+h == T and W_next >= G) else 0
        # Q-update
        s_next = (W_next, t+h)
        Q[s][k] += alpha * (r + gamma * max(Q[s_next]) - Q[s][k])
        s, W, t = s_next, W_next, t+h

2.4 Prémont(2021) 개선 — Adaptive Learning Rate

$$\alpha_k(n) = \frac{1}{1 + \text{visits}(s_k, a_k, n)}$$
Watkins-Dayan 2차 조건 충족 + 수렴 속도 2–3배 가속.

§3. Deep RL 3대 계열

VALUE
Value-Based
  • NN으로 $Q(s,a;\theta_Q)$ 근사
  • DQN · Double DQN · Dueling
  • ✓ Off-policy · sample efficient
  • ✗ Discrete action만
POLICY
Policy-Based
  • $\pi(a|s;\theta_\pi)$ 직접 학습
  • REINFORCE · TRPO
  • ✓ Continuous · stochastic
  • ✗ High variance
A-C
Actor-Critic
  • Actor + Critic
  • A2C · PPO · SAC · TD3
  • ✓ Low variance · continuous
  • ✗ 구현 복잡

3.1 PPO (Proximal Policy Optimization)

Schulman et al. (2017). RL 실무의 default choice.
$$L^{\text{CLIP}}(\theta) = \mathbb{E}_t\!\left[ \min(r_t(\theta) A_t,\ \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) A_t) \right]$$
$r_t(\theta) = \pi_\theta / \pi_{\theta_{\text{old}}}$: importance ratio. $\epsilon \approx 0.2$: clip 범위. Trust region 제약으로 large update 방지.

3.2 SAC (Soft Actor-Critic)

Haarnoja et al. (2018). Off-policy + continuous action.
$$J(\pi) = \mathbb{E}\!\left[ \sum_t r_t + \alpha_H \mathcal{H}(\pi(\cdot|s_t)) \right]$$
Entropy 보너스 $\mathcal{H}(\pi)$ 추가로 탐험을 보상에 내재화.
언제 PPO
  • Discrete or simple continuous
  • Robustness 우선
  • ★ Bauman(2024) 선택
언제 SAC
  • Complex continuous
  • Sample efficiency 중요
  • Off-policy

§4. PPO 구현 — Stable-Baselines3 + Gym

4.1 GBWM Environment (OpenAI Gym API)

import gymnasium as gym
import numpy as np

class GBWMEnv(gym.Env):
    def __init__(self, W0=100, G=200, T=10, K=15):
        self.action_space = gym.spaces.Discrete(K)
        self.observation_space = gym.spaces.Box(
            low=np.array([0, 0]), high=np.array([1000, T]),
            dtype=np.float32)
        self.W0, self.G, self.T, self.K = W0, G, T, K
        self.mus = np.linspace(0.02, 0.10, K)
        self.sigs = np.linspace(0.05, 0.20, K)

    def step(self, action):
        k = int(action)
        mu, sig = self.mus[k], self.sigs[k]
        Z = self.np_random.standard_normal()
        self.W *= np.exp((mu - sig**2/2) + sig * Z)
        self.t += 1
        terminated = (self.t >= self.T)
        reward = float(self.W >= self.G) if terminated else 0.0
        return self._get_obs(), reward, terminated, False, {}

§5. Bauman (2024) — HMM Regime-Switching DRL

Bauman, Gašperov, Goluža, Kostanjčar (2024) NLDL 2024. University of Zagreb 연구팀.

5.1 HMM Regime Detection

Hidden states $X_t \in \{1, \ldots, R\}$, $R=2$ (bull, bear):
$$Y_t | X_t \sim \mathcal{N}(\mu_{X_t}, \Sigma_{X_t})$$
Baum-Welch (EM)로 훈련. Forward-backward로 $P(X_t | Y_{1:t})$ 추정:
$$\hat{p}_r(t) = P(X_t = r | Y_{1:t}) \propto P(Y_t|X_t=r) \sum_{r'} P(X_t=r|X_{t-1}=r') \hat{p}_{r'}(t-1)$$

5.2 State Augmentation (핵심 혁신)

AUGMENTED STATE
$$s_t = (W_t, t, \hat{p}_{\text{bull}}, \hat{p}_{\text{bear}})$$
HMM regime 확률을 state에 직접 통합. Policy $\pi(a | W, t, \hat{p})$가 regime에 조건부로 반응.

§6. Dixon-Halperin G-Learner · GIRL — 대안 접근

6.1 핵심 아이디어

Entropy-regularized RL + LQR 연결:
$$V(s) = \max_a \mathbb{E}[r + \gamma V(s')] - \lambda D_{KL}(\pi_\theta(\cdot|s) \| \pi_0(\cdot|s))$$
Quadratic reward + Gaussian reference → LQR 구조, closed-form 가능.

6.2 LQR Connection의 의미

G-Learner 장점
  • 해석 가능성 ↑
  • 계산 복잡도 ↓
  • 100+ assets 확장
  • 규제 당국 설득 용이
GIRL — Inverse RL
  • 역사적 거래 관찰
  • 투자자 implicit goal 추출
  • ISP 자동 작성
  • 행동재무 관점 결합

§7. 실무 진입의 3대 장벽

1
해석 가능성 (Interpretability)
문제: Deep NN policy는 black box. "왜 이 결정?" 설명 불가. ISP 문서 기록 불가능.
해결: Model distillation · SHAP/LIME · Policy heatmap · Hybrid (G-Learner).
2
규제 (Regulation)
한국 환경: 금감원 혁신금융서비스 지정, 자문·일임업 규제, 상품 "어떻게 작동"설명 의무.
RL 이슈: 학습 데이터 대표성, OOD 동작 보장, model drift, accountability.
3
Robustness (견고성)
OOD 우려: 학습 1871–1991, 실행 2020–2030(금리0, 팬데믹, 지정학). Catastrophic failure 가능성.
검증: Stress testing(GFC·COVID·bond crash) · Adversarial · Domain randomization · Fat-tail MC.
